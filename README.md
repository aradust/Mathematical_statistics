# Mathematical_statistics
Учебные задачки с курса математической статистике по проверке гипотез
Постановка задач:
## Задание 1

Используя
нормальную выбороку из файла `sample_1.csv`:
1. Найдите точные доверительные интервалы (ДИ) для среднего значения:
   - Когда дисперсия известна.
   - Когда дисперсия неизвестна.
2. Найдите точные доверительные интервалы для дисперсии:
   - Когда среднее значение известно.
   - Когда среднее значение неизвестно.

Исходные параметры для расчетов возьмите из файла `parameters.txt`.
Доверительные интервалы необходимо построить используя: первые 50, первые 100, первые 500, первые 1000, первые 5000 и все элементы выборки.

## Задание 2

Используя данные из файла `sample_2.csv`:
Проведите проверку гипотезы (еайдите p-value) о том, что выборка равномерно распределена на интервале [0; 1], с использованием:
   - Критерия Колмогорова.
   - Критерия хи-квадрат.

Для каждого критерия необходимо найти по 3 p-value, используя: первые 50, первые 100, все элементы выборки.

## Задание 3

Используя нормальные выбороки из файлов `sample_3_1.csv` и `sample_3_2.csv` проверьте гипотезу о равенстве дисперсий двух выборок.

## Задание 4

Используя  нормальные выбороки из файлов `sample_4_1.csv` и `sample_4_2.csv` проверьте гипотезу о равенстве средних значений двух выборок при условии, что дисперсии равны.
